Stadt Land Fluss Spielanleitung

Dieses Spiel wird Ihnen von https://stadtlandfluss.cool/ bereitgestellt. Wenn Sie einem Spiel beitreten oder ein Spiel erstellen, werden Sie innerhalb dieses Fensters weitergeleitet. Dort gelten die Rechte und Nutzungsbedingungen des Anbieters.

Um einem Spiel beitreten zu können, muss ein Mitglied ihrer Spielgruppe ein Spiel erstellt haben.

Den Link zum Spiel finden Sie im Chat der Spielgruppe, nachdem der Ersteller des Spiels ihn dort veröffentlicht hat.

Kopieren Sie den Link zum Spiel und fügen Sie ihn in das folgende Eingabefeld ein.

Beitreten
Ungültige Eingabe!

Drücken Sie auf die Beitreten-Schaltfläche, um das Spiel zu starten.

Um ein Spiel zu erstellen, müssen Sie auf Stadt Land Fluss ein privates Spiel erstellen.

Kopieren Sie den Link anschließend in den Chat ihrer Spielgruppe, damit die anderen Spieler ihrem Spiel beitreten können.

Ein privates Spiel erstellen Sie, indem Sie auf die folgende Erstellen-Schaltfläche klicken.

Innerhalb des Spiels müssen Sie auf "Spiel erstellen" klicken. Den Link kopieren Sie, indem Sie auf "Kopieren" klicken.

Erstellen

Die Spielfläche können Sie vergrößern, indem Sie auf das Symbol mit den drei vertikalen Strichen im Zentrum ihres Sichtbereichs klicken und die Taste gedrückt halten.

Verschieben Sie die Maus auf der horizontalen Achse.

Lösen Sie den Klick, sobald Sie die optimale Position erreicht haben.
